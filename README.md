Программа написана на Visual Studio, соответственно открыть можно через sln-файл. exe-файл находиться по пути Lab3.4W\x64\Debug\.
Функционал программы:
Добавление  клиента. Открывается окно, в которую нужно ввести имя клиента и его номер
Добавление  тарифа. Открывается окно, в котором нужно выбрать тип груза и назначить за него цену
Изменение данных. Нужно выбрать из списка клиента или тариф и нажать на кнопку. Откроется окно, в котором можно изменить параметры клиента или тарифа
Удаление клиента. Нужно выбрать из списка клиента и нажать на кнопку. Клиент удаляется безвозратно
Удаление тарифа. Нужно выбрать из списка тарифа и нажать на кнопку. Тариф удаляется безвозратно
Загрузить. Загружается таблица клиентов из csv-файла. Загружаемый файл должен быть файлом, полученным при созранении таблицы в программе
Созранить. Сохраняется таблица в csv-файл по указанному пути.
Сортировка. Работает сортировка в самой таблице по названиям столбцов.
